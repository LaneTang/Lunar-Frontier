# Lunar Frontier 脑暴笔记

> 这不是设计文档。这是一堆还在发酵的想法。很多可能是错的，很多会变。

- **项目代号：** 月球开拓者 (Lunar Frontier)
- **大概方向：** 硬科幻 / 模拟经营 / 基地建设 + 工厂自动化
- **对标：** 戴森球计划、缺氧、纪元1800、冰汽时代2
- **时间预期：** 半年到一年的长线项目

---

## 为什么要做这个

游戏不是效率模拟器，是文明推演器。人类即将登上月球和火星了，对未来文明形态的预测与推演很重要。游戏是一个很好的传达这个建构未来的媒介。而且我自己也喜欢玩这类工厂模拟、经营模拟游戏。

电影可以展示一种未来，但只有游戏可以让玩家亲手参与未来的建构。

---

## 一些还在想的问题

- 玩家到底是谁？Moontopia 文明发展理事会？一个群体而不是一个人？
- "平台"模型到底对不对——为什么各家势力不自己建？南极不就是各建各的吗？
- 核心循环到底是什么？"进口 vs 自产"的决策模板够不够撑起整个游戏？
- 理事会投票怎么运作？玩家直接决定还是需要多数票？
- 势力的"需求"是什么形式——定期任务？持续条件？里程碑？

---

## 零散想法

月球不是目的地，月球是跳板。码头、船坞、加油站。质量投射器不是奇观，是逻辑终点。

物流即货币。地月运输不是效率问题，是生存问题。每一次发射都价值连城。

三阶段：求生（我会不会死）→ 商业化（我怎么赚钱）→ 宏伟工程（我怎么统筹全月球的产能）。每个阶段的核心焦虑自然转移。

一分钟体验：地球又送上来一批开拓者，供氧不够了。A：进口氧气（快、贵、占配额）。B：扩建制氧厂（慢、但长期可持续）。这个决策就是核心心跳。

两个系统互相拉扯：生命维持系统（硬约束——氧气、水、食物、电力、温度）vs 理事会政治系统（软约束——各方势力的期望）。两个系统在争夺同一个东西：发射配额。

Moontopia 是第一个到的，建了基础设施，后来者来进驻。像中国建了空间站，其他国家来进驻。但 Moontopia 是自治组织，不是国家操控的。

起源可以像群星捏物种一样自定义——国际联合体、私人公司、科研联盟、慈善基金。不同起源不同开局。

势力 = 满意度 + 需求 + 回报。NASA 满意度高 → 续租 → 给经费。满意度低 → 撤出。和冰汽时代 2 的关键区别：势力是客户，不是人民，不满意可以走。

纪元 1800 的核心教训：需求驱动扩张，不是你想建什么，是需求倒逼你建什么。

冰汽时代 2 的核心教训：从微观到宏观的视角转换，议会系统，区域群聚效应。

## 现实里正在发生的事

- **GRU Space**：22 岁华裔少年，2 个人的公司，Y Combinator 投资，收月球酒店定金。计划 2032 年开业。
- **SpaceX**：战略重心从火星转向月球，"自生长城市"，10 年内建成。
- **NASA**：$200 亿直接建月面基地，三阶段（2025-2032+），裂变反应堆 + 太阳能。
- **中国 ILRS**：国际月球科研站，三阶段（2028-2050），选址南极。
- **ESA**：月球村概念，多国共建。

建立真实月球城市需要什么（三层）：
1. 生存基础：电力、氧气、水、食物、温度、辐射防护
2. 运行基础：着陆场、通信、居住舱、运输、机器人
3. 发展基础：ISRU 工厂、科研、工业、商业、发射加注
